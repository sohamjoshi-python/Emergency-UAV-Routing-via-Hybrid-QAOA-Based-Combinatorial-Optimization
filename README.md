# Quantum Optimization for UAV Routing

This repository contains the experimental code and analysis for evaluating **classical and quantum optimization approaches for Unmanned Aerial Vehicle (UAV) routing problems**. The routing problem is formulated as an **Integer Linear Program (ILP)** and converted into a **Quadratic Unconstrained Binary Optimization (QUBO)** model suitable for quantum optimization algorithms.

The experiments compare classical optimization with quantum formulations across varying problem sizes using transportation network data.

# File Descriptions

## PhoenixAllAlgorithmsRun.ipynb

Main experiment notebook.

This notebook performs the full experimental pipeline:

- Builds the transportation network
- Generates vehicle and request scenarios
- Constructs the routing graph
- Formulates the ILP routing problem
- Converts the ILP to a QUBO model
- Runs the classical and quantum optimization approaches
- Saves the experiment results

The output of this notebook is saved to **data.csv**.

---

## ClassicalAlgorithm2D.ipynb

Implementation of the **classical baseline routing algorithm** used for comparison.

This notebook contains:

- The 2D routing model
- Classical trip assignment logic
- Baseline optimization solver
- Supporting utilities used in the experiments

---

## AnalyzeOutput.ipynb

Analysis and visualization notebook.

This notebook:

- Loads `data.csv`
- Performs statistical analysis
- Generates plots and scaling graphs
- Compares performance between optimization methods

Example metrics analyzed include:

- Percent of requests serviced
- Average waiting time
- Maximum waiting time
- Average detour factor
- Vehicle miles traveled (VMT)
- QUBO size and scaling behavior

---

## data.csv

This file contains the **raw experimental results** generated by running `PhoenixAllAlgorithmsRun.ipynb`.

Each row corresponds to a single experiment trial and includes metrics such as:

- Number of vehicles
- Number of requests
- Algorithm type
- Service rate
- Waiting times
- Detour factors
- Runtime measurements
- QUBO statistics

# Running the Experiments

### 1. Install dependencies

```bash
pip install numpy pandas networkx matplotlib scipy dimod dwave-system qiskit
```

### 2. Run the experiment notebook
Open and run
```
PhoenixAllAlgorithmsRun.ipynb
```
This notebook generates routing scenarios, runs the algorithms, and outputs results to data.csv.
### 3. Running the Analysis
After the experiments finish, run:
```
AnalyzeOutput.ipynb
```
This notebook generates visualizations and statistical comparisons of the results.
